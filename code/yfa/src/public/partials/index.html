node.js nodejs.org »

Node.js® is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications.

Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices.

This application and the accompanying book introduce beginners to the world of API development using node.js.

Express.js expressjs.com »

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

With a myriad of HTTP utility methods and middleware at your disposal, creating a robust API is quick and easy.

Almost nothing is coded 'from scratch' these days. Express.js provides a quick and maintainable framework for creating APIs and traditional server-side rendered web applications.

MongoDB mongodb.org »

MongoDB (from "humongous") is an open-source document database, and the leading NoSQL database. Written in C++.

MongoDB is used with mongoose.js to create a structured and schemaless application model in the database while enforcing common schema-like constraints in the application. This gives us an almost class-like structure within node.js, helping to eliminate some of the shock of moving from a classical language like Java or C# to a prototypical language like JavaScript.

AngularJS angularjs.org »

HTML is great for declaring static documents, but it falters when we try to use it for declaring dynamic views in web-applications. AngularJS lets you extend HTML vocabulary for your application. The resulting environment is extraordinarily expressive, readable, and quick to develop.

This application is written with AngularJS on the client-side to demonstrate reusable, declarative, and behavior-driven code. AngularJS answers a lot of problems with static HTML by providing a framework for extending HTML structure and behavior. AngularJS offers a convention to tame large-scale web applications in a way that many other frameworks can't achieve without a steady developer discipline.
